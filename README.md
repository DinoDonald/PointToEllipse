# PointToEllipse
Orthogonal Projection of a Point onto an ellipse

In order to get the orthogonal projection of a point onto an ellipse
We must solve the equation

$$(\textbf{x}(t) - \textbf{p})\cdot \textbf{x}'(t) = 0$$
